# 8-Recursion

Project to find out what is recursion and how to implement it on simple problems. 
Every function will be using recursivity:

* 0: puts recursion: display a string
* 1: print rev recursion: string in reverse
* 2: strlen: lenght of string
* 3: factorial: factorial of a given number
* 4: pow: given power of a given number
* 5: sqrt: square root of the number
* 6: is prime: find out if the number is a prime
* 7: is palindrome: find out if the word is palindrome
