# Low Level Programming
## C language

* 0x00: understanding the steps of compilation and display of strings. 
* 0x01: Use of variables, arithmetic operators as well as statements.
* 0x02: Use of functions, prototypes, headers files and loops.
* 0x03: Debugging programs.
* 0x04: More functions, prototypes, loops and nested loops.
* 0x05: Pointers, arrays and strings.
* 0x07: Pointers, arrays and strings.
